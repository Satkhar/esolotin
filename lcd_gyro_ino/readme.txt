проект разработанный в среде arduinoIDE, на базе arduinoUNO.
задача проекта - получение навыков работы в среде разработки arduinoIDE.

главное обрабатывающее устройство - плата arduino.

функционал:
измерение встроенным АЦП значение сопротивления;
опрос гироскопа adxl345;
отрисовка полученных значений на LCD экране ST7735S;
переключение в режим низкого энергопотребления по нажатию кнопки.

опрос гироскопа производится по протоколу I2C, вывод изображения на экран производится по протоколу SPI.
